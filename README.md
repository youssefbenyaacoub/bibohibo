# Bibicha - 22nd Birthday Experience

A premium, interactive Sims 4-themed birthday web application created for Bibicha's 22nd birthday.

## ✨ Features
- **Interactive Welcome**: Pop 22 balloons to unlock the experience (with "Heat Waves" music).
- **Town Hub**: Explore a custom-styled map with Sim-like navigation.
- **Memory Gallery**: A beautiful photo book with "Talking to the Moon" by Bruno Mars.
- **Gift Scene**: A 3D Minecraft-themed gift box reveal with "As It Was" by Harry Styles.
- **Finale Scene**: A stunning 3D cake with magical particle effects and a personalized message.

## 🚀 How to Run Locally
1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠️ Deployment (GitHub Pages)
To deploy this project to GitHub:
1. Create a new repository on GitHub named `bibicha-birthday`.
2. Run the following commands in your terminal:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Bibicha Birthday Experience"
   git branch -M main
   git remote add origin https://github.com/<your-username>/bibicha-birthday.git
   git push -u origin main
   ```
